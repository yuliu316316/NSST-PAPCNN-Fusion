This package contains the MATLAB code which is associated with the following paper:

Ming Yin, Xiaoning Liu, Yu Liu*, Xun Chen "Medical Image Fusion With Parameter-Adaptive Pulse Coupled Neural Network in Nonsubsampled Shearlet Transform Domain", IEEE Transactions on Instrumentation and Measurement, in press, 2018.
https://ieeexplore.ieee.org/document/8385209/

Edited by Yu Liu.   

Usage of this code is free for research purposes only. 

Please refer to the above paper if you use this code. Thank you.

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

The demo files are:
Script_gray.m for the fusion of two gray-scale images
Script_color.m for the fusion of one gray-scale (anatomical) image and one color (functional) image.
